Zig –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –∫–∞–∫ –∑–Ω–∞–∫–æ–≤—ã–µ, —Ç–∞–∫ –∏ –±–µ–∑–∑–Ω–∞–∫–æ–≤—ã–µ —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã–µ —Ç–∏–ø—ã —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç–µ–π. –û–Ω–∏ –æ–ø—Ä–µ–¥–µ–ª—è—é—Ç—Å—è —Å –ø–æ–º–æ—â—å—é iN (–∑–Ω–∞–∫–æ–≤—ã–µ) –∏ uN (–±–µ–∑–∑–Ω–∞–∫–æ–≤—ã–µ), –≥–¥–µ N ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –±–∏—Ç –≤ —á–∏—Å–ª–µ. –¢–∞–∫ –Ω–∞–ø—Ä–∏–º–µ—Ä `u32` –æ–±—ä—è–≤–ª–µ–Ω–∏–µ —Ç–∏–ø–∞ —É–∫–∞–∑—ã–≤–∞–µ—Ç, —á—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ, —Å –∫–æ—Ç–æ—Ä—ã–º –æ–Ω–æ —Å–≤—è–∑–∞–Ω–æ, –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å —Ü–µ–ª—ã–º —á–∏—Å–ª–æ–º –±–µ–∑ –∑–Ω–∞–∫–∞ (—Ç–∏–ø—ã —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª —Å–æ –∑–Ω–∞–∫–æ–º –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å i –≤–º–µ—Å—Ç–æ u), –∫–æ—Ç–æ—Ä–æ–µ –∑–∞–Ω–∏–º–∞–µ—Ç 32 –±–∏—Ç–∞ –ø–∞–º—è—Ç–∏. –ü–æ–º–∏–º–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö –ø—Ä–∏–≤—ã—á–Ω—ã—Ö –≤—Å–µ–º —Ç–∏–ø–æ–≤ –¥–∞–Ω–Ω—ã—Ö —Ç–∞–∫–∏—Ö –∫–∞–∫ `u8`, `i8`, `u16`, `i16`, `u32`, `i32`, `u64`, `i64`, `u128` –∏ `i128` Zig —Ç–∞–∫–∂–µ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Ç–∏–ø—ã `u1`, `i1`, `u2`, `i2`, `u4`, `i4`, `i47`, `u47`, —Ç.–µ. Zig —É–º–µ–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å —Å —Ü–µ–ª—ã–º–∏ —á–∏—Å–ª–∞–º–∏ –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–π –¥–ª–∏–Ω—ã (–≤ –±–∏—Ç–∞—Ö). –ö–æ–Ω–µ—á–Ω–æ, –≤–∞–º, –≤–æ–∑–º–æ–∂–Ω–æ, —Ç–∞–∫–∏–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö –Ω–∏–∫–æ–≥–¥–∞ –Ω–µ –ø–æ–Ω–∞–¥–æ–±—è—Ç—Å—è, –Ω–æ, –≤–µ—Ä–æ—è—Ç–Ω–æ, –µ—Å—Ç—å —Å–ª—É—á–∞–∏, –∫–æ–≥–¥–∞ —ç—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–æ–ª–µ–∑–Ω–æ.

–ü—Ä–∏–º–µ—Ä—ã —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã—Ö —Ç–∏–ø–æ–≤:

```zig
const a: i32 = -100; // –ó–Ω–∞–∫–æ–≤–æ–µ 32-–±–∏—Ç–Ω–æ–µ —á–∏—Å–ª–æ
const b: u8 = 255;   // –ë–µ–∑–∑–Ω–∞–∫–æ–≤–æ–µ 8-–±–∏—Ç–Ω–æ–µ —á–∏—Å–ª–æ
```

–ö–∞–∂–¥—ã–π –≤–∞—Ä–∏–∞–Ω—Ç —Å–æ –∑–Ω–∞–∫–æ–º –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Å–ª–∞ –æ—Ç $ -2 ^ {n - 1} $ –¥–æ $ 2 ^ {n - 1} - 1 $ –≤–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ, –≥–¥–µ n ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –±–∏—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —ç—Ç–æ—Ç –≤–∞—Ä–∏–∞–Ω—Ç. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, i8 –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Å–ª–∞ –æ—Ç $ -(2 ^ 7 ) $ –¥–æ $ 2 ^ 7 - 1 $, —á—Ç–æ —Ä–∞–≤–Ω–æ –∑–Ω–∞—á–µ–Ω–∏—è–º –æ—Ç -128 –¥–æ 127. –í–∞—Ä–∏–∞–Ω—Ç—ã –±–µ–∑ –∑–Ω–∞–∫–∞ –º–æ–≥—É—Ç —Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Å–ª–∞ –æ—Ç 0 –¥–æ $ 2 ^ n - 1 $, –ø–æ—ç—Ç–æ–º—É u8 –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Å–ª–∞ –æ—Ç 0 –¥–æ $ 2 ^ 8 - 1 $, —á—Ç–æ —Ä–∞–≤–Ω–æ –∑–Ω–∞—á–µ–Ω–∏—è–º –æ—Ç 0 –¥–æ 255.

–í —è–∑—ã–∫–µ Zig —Ç–∞–∫–∂–µ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω–æ-–∑–∞–≤–∏—Å–∏–º—ã–µ —á–∏—Å–ª–æ–≤—ã–µ —Ç–∏–ø—ã ‚Äî `usize` (–±–µ–∑–∑–Ω–∞–∫–æ–≤—ã–π) –∏ `isize` (–∑–Ω–∞–∫–æ–≤—ã–π). –≠—Ç–∏ —Ç–∏–ø—ã –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω—ã –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å —Ä–∞–∑–º–µ—Ä–∞–º–∏, –∏–Ω–¥–µ–∫—Å–∞–º–∏ –∏ –∞–¥—Ä–µ—Å–∞–º–∏ –ø–∞–º—è—Ç–∏, –ø—Ä–∏—á–µ–º –∏—Ö —Ä–∞–∑–º–µ—Ä –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç–∏ –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã –∫–æ–º–ø—å—é—Ç–µ—Ä–∞, –Ω–∞ –∫–æ—Ç–æ—Ä–æ–º –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –ø—Ä–æ–≥—Ä–∞–º–º–∞:

* –ù–∞ 64-–±–∏—Ç–Ω–æ–π –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–µ (x86_64, ARM64) usize –∏ isize –±—É–¥—É—Ç 64-–±–∏—Ç–Ω—ã–º–∏ (8 –±–∞–π—Ç).
* –ù–∞ 32-–±–∏—Ç–Ω–æ–π –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–µ (x86, ARM32) –∏—Ö —Ä–∞–∑–º–µ—Ä –±—É–¥–µ—Ç 32 –±–∏—Ç–∞ (4 –±–∞–π—Ç–∞).
* –ù–∞ 16-–±–∏—Ç–Ω—ã—Ö –∏–ª–∏ –¥—Ä—É–≥–∏—Ö —Å–∏—Å—Ç–µ–º–∞—Ö –∏—Ö —Ä–∞–∑–º–µ—Ä –±—É–¥–µ—Ç —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç–∏ –∞–¥—Ä–µ—Å–∞—Ü–∏–∏ –ø–∞–º—è—Ç–∏.

–¢–∏–ø—ã `usize` –∏ `isize` –¥–µ–ª–∞—é—Ç –∫–æ–¥ –±–æ–ª–µ–µ –ø–æ—Ä—Ç–∏—Ä—É–µ–º—ã–º –∏ –æ–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–º –¥–ª—è –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–π –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã:

```zig
var a: usize = 10;  // –ù–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–µ —Å –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–æ–π x86_x64 —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç—å —Ç–∏–ø–∞ –±—É–¥–µ—Ç 64 –±–∏—Ç–∞
```

–í Zig –Ω–µ—Ç –ø–æ–¥–¥–µ—Ä–∂–∫–∏ –ª–∏—Ç–µ—Ä–∞–ª—å–Ω—ã—Ö —Å—É—Ñ—Ñ–∏–∫—Å–æ–≤, –∫–∞–∫ –≤ –¥—Ä—É–≥–∏—Ö —è–∑—ã–∫–∞—Ö –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è, —Ç–∞–∫–∏—Ö –∫–∞–∫ C –∏–ª–∏ C++. –í—ã –≤—Å–µ–≥–¥–∞ –¥–æ–ª–∂–Ω—ã —è–≤–Ω–æ —É–∫–∞–∑—ã–≤–∞—Ç—å —Ç–∏–ø –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –ª–∏–±–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø—Ä–∏–≤–µ–¥–µ–Ω–∏–µ —Ç–∏–ø–æ–≤ –≤—Ä—É—á–Ω—É—é. –û–¥–Ω–∞–∫–æ –≤ Zig –µ—Å—Ç—å –ø–æ–¥–¥–µ—Ä–∂–∫–∞ –ø—Ä–µ—Ñ–∏–∫—Å–æ–≤ –¥–ª—è –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏—è —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Å–∏—Å—Ç–µ–º —Å—á–∏—Å–ª–µ–Ω–∏—è –≤ —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã—Ö –ª–∏—Ç–µ—Ä–∞–ª–∞—Ö (0x, 0o, 0b). –†–∞—Å—Å–º–æ—Ç—Ä–∏–º –ø—Ä–∏–º–µ—Ä:


–ß–∏—Å–ª–æ–≤—ã–µ –ª–∏—Ç–µ—Ä–∞–ª—ã —Ç–∞–∫–∂–µ –º–æ–≥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `_` –≤ –∫–∞—á–µ—Å—Ç–≤–µ –≤–∏–∑—É–∞–ª—å–Ω–æ–≥–æ —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—è –¥–ª—è –æ–±–ª–µ–≥—á–µ–Ω–∏—è —á—Ç–µ–Ω–∏—è —á–∏—Å–ª–∞, –Ω–∞–ø—Ä–∏–º–µ—Ä 1_000_000, –∫–æ—Ç–æ—Ä—ã–π –±—É–¥–µ—Ç –∏–º–µ—Ç—å —Ç–∞–∫–æ–µ –∂–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–∞–∫ –µ—Å–ª–∏ –±—ã –±—ã–ª–æ –∑–∞–¥–∞–Ω–æ 1000000.

```zig

```

–ö–æ–≥–¥–∞ –í—ã —Ä–∞–±–æ—Ç–∞–µ—Ç —Å —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã–º–∏ —Ç–∏–ø–∞–º–∏ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–π —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç–∏, –≤–∞–∂–Ω–æ —É—á–∏—Ç—ã–≤–∞—Ç—å, —á—Ç–æ –º–æ–∂–µ—Ç –ø—Ä–æ–∏–∑–æ–π—Ç–∏ —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–µ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏–µ. –î–æ–ø—É—Å—Ç–∏–º, –∏–º–µ–µ—Ç—Å—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è —Ç–∏–ø–∞ `u8`, –∫–æ—Ç–æ—Ä–∞—è –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç 0 –¥–æ 255. –ï—Å–ª–∏ –ø–æ–ø—ã—Ç–∞—Ç—å—Å—è –∏–∑–º–µ–Ω–∏—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –Ω–∞ –∑–Ω–∞—á–µ–Ω–∏–µ –≤–Ω–µ —ç—Ç–æ–≥–æ –¥–∏–∞–ø–∞–∑–æ–Ω–∞, –Ω–∞–ø—Ä–∏–º–µ—Ä, 256, –ø—Ä–æ–∏–∑–æ–π–¥—ë—Ç —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–µ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏–µ. –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –æ–±—ã—á–Ω—ã–µ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, `+`, `-`, `*`, `/`) –≤—ã–∑—ã–≤–∞—é—Ç –æ—à–∏–±–∫—É –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ –∏–ª–∏ –ø–∞–Ω–∏–∫—É –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è, –µ—Å–ª–∏ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏–µ. –û–¥–Ω–∞–∫–æ Zig —Ç–∞–∫–∂–µ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –æ–±–µ—Ä—Ç–æ—á–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ (`%=`) –∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –Ω–∞—Å—ã—â–µ–Ω–∏–µ–º (`+|=`, `-|=`, `*|=`) –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –ø–æ–≤–µ–¥–µ–Ω–∏–µ–º –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏—è.

–ü—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –æ–±—ã—á–Ω—ã—Ö –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –ø–æ–≤–µ–¥–µ–Ω–∏–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —Ç–æ–≥–æ, –≤ –∫–∞–∫–æ–º —Ä–µ–∂–∏–º–µ —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω–∞ –≤–∞—à–∞ –ø—Ä–æ–≥—Ä–∞–º–º–∞. –ï—Å–ª–∏ –≤—ã —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–ª–∏ –ø—Ä–æ–≥—Ä–∞–º–º—É –≤ —Ä–µ–∂–∏–º–µ debug, —Ç–æ –ø—Ä–∏ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏–∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤—ã–∑–æ–≤—É—Ç –æ—à–∏–±–∫—É –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ –∏–ª–∏ –ø–∞–Ω–∏–∫—É –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è. –í —Ä–µ–∂–∏–º–µ release –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –º–æ–∂–µ—Ç –æ–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞—Ç—å –∫–æ–¥, –∏ –ø–æ–≤–µ–¥–µ–Ω–∏–µ —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –Ω–µ–æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω—ã–º.
